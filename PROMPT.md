# Промпт для создания приложения "Генератор случайных звуков"

Этот промпт предназначен для использования в ИИ-агрегаторах или LLM (например, Claude, GPT-4) для создания полностью функционального веб-приложения.

---

**Задача:** Создай одностраничное веб-приложение на React (Next.js App Router, TypeScript), которое генерирует случайные звуковые эффекты в реальном времени.

**Технические требования:**
1.  **Стек:** Next.js (последняя версия), React, TypeScript, Tailwind CSS (для стилизации).
2.  **Звук:** Используй **Web Audio API**. Приложение НЕ должно использовать внешние аудиофайлы; все звуки должны синтезироваться программно с помощью осцилляторов (`OscillatorNode`).
3.  **Функциональность:**
    *   Кнопка "Сгенерировать звук", которая при нажатии создает короткий случайный звук.
    *   Случайный выбор формы волны (`sine`, `square`, `sawtooth`, `triangle`) для каждого звука.
    *   Случайный выбор частоты в заданном диапазоне (например, 100 Гц - 2000 Гц).
    *   Плавное нарастание и затухание громкости (ADSR огибающая: быстрый attack и decay), чтобы избежать щелчков.
    *   **Авто-режим:** Чекбокс/переключатель, который при включении генерирует звуки через случайные промежутки времени (от 0.5 до 2 секунд).
    *   **Настройки (UI):**
        *   Слайдер регулировки общей громкости.
        *   Слайдеры для настройки диапазона частот (Min/Max).
4.  **Дизайн:**
    *   Простой, чистый и современный интерфейс.
    *   Используй темную или светлую тему (можно адаптивную).
    *   Центрированный макет с карточкой управления.
    *   Визуальный фидбек при воспроизведении звука (например, анимация кнопки или индикатор).

**Структура кода:**
*   Весь код должен быть в одном файле `page.tsx` (или разделен на компоненты в той же директории).
*   Используй хуки `useRef` для хранения `AudioContext` и `useState` для управления состоянием интерфейса.
*   Обязательно обработай инициализацию `AudioContext` после первого взаимодействия пользователя (требование браузеров).

**Результат:** Предоставь полный исходный код файла `page.tsx`, готовый к копированию и запуску в проекте Next.js.
